# ccPaper
RUNNING CC 静电场仿真平台项目介绍

项目概述
RUNNING CC 是一个基于物理优先的机器学习架构的静电场仿真平台，专为科学研究和教育用途设计。这个平台集成了多种电荷模型、高性能计算引擎、先进的可视化技术和机器学习加速功能，为用户提供直观且功能强大的电场分析工具。

核心功能
1. 多种物理模型支持
平台支持多种电荷分布模型的电场计算：
- 点电荷模型：基础电荷模型，精确计算点电荷周围的电场分布
- 电偶极子模型：模拟正负电荷对形成的电场
- 有限长线电荷模型：支持任意位置、任意方向的导线电荷计算，提供解析解
- 圆环电荷模型：计算带电圆环周围的电场，包括椭圆积分计算
- 边界元法(BEM)求解器：用于更复杂的导体表面电荷分布问题

2. 高性能计算引擎
- 统一接口设计：所有物理模型通过统一的抽象基类实现，保证接口一致性
- 3D优先架构：以3D计算为基础，2D作为特例处理
- 科学计算优化：利用numpy进行高效数值计算
- 自适应步长积分：提高电场线生成的精度和稳定性
- 奇异点处理：智能避免电荷附近的计算奇点

3. 高级可视化系统
- 多维度视图：支持2D和3D模式的电场可视化
- 多种可视化方式：
  - 电场矢量场显示
  - 电势分布热力图
  - 电场线绘制（自适应密度）
- 交互式探索：支持缩放、旋转等操作
- 高质量渲染：支持Matplotlib和Plotly等多种后端

4. 机器学习加速
- RBF插值器：使用径向基函数加速电场计算
- 自适应采样：智能在电荷附近生成训练数据
- 高性能预测：快速预测未计算点的电场值

使用方法
快速开始
1. 启动应用：运行 `run_app.bat` 脚本启动Streamlit应用
2. 选择模型：在左侧面板选择所需的物理模型（点电荷、线电荷、圆环电荷等）
3. 设置参数：根据所选模型调整电荷参数（大小、位置、数量等）
4. 运行仿真：点击"开始计算"按钮执行电场计算
5. 查看结果：在结果面板中查看电场线、电势分布等可视化效果

模型参数设置
- 点电荷模型：设置电荷值、位置坐标
- 电偶极子模型：设置偶极矩、方向、中心位置
- 线电荷模型：设置线电荷密度、线段端点、数量
- 圆环电荷模型：设置半径、总电荷、中心位置
- BEM求解器：设置导体几何形状、边界条件

计算与可视化选项
- 维度选择：2D或3D模式
- 网格密度：调整计算点数量
- 场线数量：控制电场线的数量和密度
- 渲染方式：选择不同的可视化后端和样式

技术架构
项目采用模块化设计，主要组件包括：
- UI层：基于Streamlit的用户界面（`ui/app.py`）
- 核心引擎：统一的计算引擎抽象（`core/engine.py`）
- 物理模型：各种电荷分布的物理实现（`physics/`目录）
- 可视化后端：支持多种渲染方式（`visualization/backends.py`）
- ML模块：机器学习加速组件（`ml/interpolator.py`）
- 数据结构：统一的数据契约和验证（`core/data_schema.py`）

特色与优势
1. 物理准确性：提供精确的解析解和高质量的数值解
2. 计算效率：结合解析公式和数值方法，平衡精度和速度
3. 用户友好：直观的界面设计，适合科研和教学
4. 可扩展性：模块化架构，易于添加新的物理模型
5. 教育价值：包含理论公式和参考文献，帮助理解电磁学概念

应用场景
- 电磁学教学演示
- 电场分布研究
- 电荷系统设计分析
- 科学论文计算支持
- 电磁学概念学习辅助
